#  Intelligent Urban Delivery Robot

## Description
A Python simulation of an intelligent delivery robot navigating
a 15x15 grid-based urban environment using 5 AI search algorithms.

## Algorithms Implemented
- Breadth First Search (BFS)
- Depth First Search (DFS)
- Uniform Cost Search (UCS)
- Greedy Best First Search
- A* Search

## Environment
- 15x15 Grid
- Roads (cost 1-5)
- Buildings (obstacles)
- Traffic Zones (cost 10-20)
- 5 Random Delivery Locations
- Live Robot Animation (tkinter GUI)

## Performance Metrics
- Path Cost
- Execution Time
- Nodes Explored

## How to Run
```bash
python delivery_robot.py
```

## Requirements
- Python 3.x
- tkinter (built-in)
